Que es Github

GitHub es una plataforma en línea basada en Git que proporciona herramientas para alojar repositorios de Git, gestionar proyectos y colaborar en código de manera eficiente. Además de alojar repositorios de código, GitHub ofrece funcionalidades como seguimiento de problemas, gestión de proyectos, integración continua y despliegue, entre otras.
